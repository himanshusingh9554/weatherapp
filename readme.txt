Weather App

Overview

This is a simple weather application built using HTML, CSS, and JavaScript. It allows users to search for a city and displays the current weather conditions, including temperature, weather description, and an icon.

Features

City Search: Users can input a city name to get weather information.
Weather Display: The app displays the current temperature, weather description, and a relevant icon.
Responsive Design: The app is designed to be responsive and work on various screen sizes.
